Fase 1 completamente cerrada. Aquí el resumen de todo lo implementado en este commit:
 
1. Caché universal de fragmentos (Azure, Polly, ElevenLabs)
Cada cliente TTS tiene ahora _cache_frags (diccionario texto→audio, máx. 5 entradas LRU):
• Salto atrás: detener() ya no borra el caché → al releer el mismo fragmento, se usa el audio ya descargado sin llamar a la API
• hablar(): prioridad caché > buffer proactivo > API
• preparar(): reutiliza caché si ya tiene el audio
• Velocidad/Volumen: al cambiar el valor, el caché se invalida automáticamente (audio obsoleto)
2. Polly 24kHz y Generative
• SampleRate="24000" añadido a synthesize_speech()
• es_nueva ahora incluye voces generative y long-form (no solo neural)
3. Etiquetas de voz coherentes en Lectura
El combo de Lectura muestra ahora el mismo formato que Ajustes:
Alba [Nueva]; Femenino; Español (España); Amazon Polly
