1A — Crear app/interfaz/ventana_proyectos.py
--------------------------------------------
Ventana independiente (wx.Frame, no modal) que permite gestionar
la jerarquía de proyectos. Se abre con Ctrl+P desde el menú.

Layout (dos paneles horizontales):

  IZQUIERDA — TreeCtrl con jerarquía:
    Nodo raíz = proyecto tipo saga / autoconclusivo / podcast / etc.
    Hijos     = libro / capítulo / episodio / etc.
    Nietos    = capítulo dentro de libro, etc.
    Cada nodo muestra: "Nombre [tipo]"
    Atajos: F2 = renombrar nodo seleccionado; Supr = eliminar

  DERECHA — Panel de detalle del nodo seleccionado:
    · Nombre        (TextCtrl editable, Intro = guardar)
    · Tipo          (Choice con los tipos: saga, libro, capitulo,
                     autoconclusivo, podcast, episodio,
                     video_youtube, guion, dialogo, otro)
    · Archivos TXT asociados (ListCtrl de solo lectura, 2 columnas:
                     nombre y ruta completa)
      Botones: "Añadir TXT..." (FileDialog) | "Quitar seleccionado"
    · Voces del proyecto (ListCtrl solo lectura, 2 columnas:
                     etiqueta y voz asignada — heredadas del padre
                     si el nodo no tiene propias)

  ABAJO — barra de botones:
    "Nuevo proyecto raíz" | "Nuevo hijo del seleccionado" |
    "Eliminar proyecto" | "Cerrar"

Requisitos de accesibilidad NVDA (OBLIGATORIO en todos los
controles nuevos):
  - SetHelpText() descriptivo en cada control
  - Botones con texto explícito (no solo iconos)
  - Al seleccionar un nodo en el árbol → foco pasa al campo Nombre
  - Confirmación wx.MessageDialog antes de eliminar
  - Al cerrar → foco vuelve al Frame principal

API de GestorProyectos que debes usar:
  from app.motor.gestor_config import GestorProyectos
  g = GestorProyectos()
  g.crear_proyecto(nombre, tipo, padre_id=None) → dict
  g.eliminar_proyecto(proyecto_id) → bool
  g.asociar_archivo(archivo_path, proyecto_id)
  g.desasociar_archivo(archivo_path)
  g.obtener_proyecto_de_archivo(archivo_path) → dict | None
  g.obtener_voces_heredadas(proyecto_id) → dict
  g.guardar_voces_proyecto(proyecto_id, voces_dict)
  g.listar_proyectos() → list[dict]  ← lista plana con 'padre_id'
  g.obtener_hijos(proyecto_id) → list[dict]

Tipos válidos (para el Choice):
  saga, libro, capitulo, autoconclusivo, podcast, episodio,
  video_youtube, guion, dialogo, otro

Prevención de doble ventana:
  En VentanaPrincipal, guardar referencia a la ventana de proyectos.
  Si ya está abierta, traerla al frente con .Raise() en lugar de
  crear una segunda instancia.
