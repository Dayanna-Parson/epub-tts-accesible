# Epub TTS Accesible

Aplicación de texto a voz accesible para libros EPUB y archivos TXT.
Diseñada para usuarios de lectores de pantalla como NVDA.

## Características

- Lectura de EPUB y TXT con voces de alta calidad
- Compatible con NVDA y otros lectores de pantalla
- Proveedores de voz: Microsoft Azure TTS, Amazon Polly, ElevenLabs y SAPI5 (sin coste)
- Gestión de proyectos y capítulos
- Grabación de audio por capítulos
- Marcadores, búsqueda y navegación accesible
